Use these symbols to build the flowchart:

AFL characterflowchart symbol
( terminal
[ process
/ i/o
< decision
| predefined
~ document
{ storage
* comment
$ manual input
` manual operation
@ display

An AFL file is a text (UTF-8) file composed of many symbol lines. Each symbol line is in the following format: 

S  _id  Label  - Output  > _to

where:

S is the symbol, as shown in the chart above

_id is an optional ID for the symbol (used to link another symbol to this one)

Label is an optional label for the symbol

Output is an optional label for the outgoing flow line

_to is an optional ID specifying the destination for the outgoing flow

 

If no outgoing flow is given, it defaults to the following symbol line.

Use ">" without an ID to specify that there is no outgoing flow.

Label and Output may contain spaces.

A word starting with an underscore _ is treated as an ID.

To use a symbol or word beginning with an underscore in a string, escape the character with a backslash.
